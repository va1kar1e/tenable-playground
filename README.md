# Tenable Playground

Playground for Testing Nessus Tenable.SC and Tenable.IO APIs

## Environment File

Create file `.env` in root directory

```plaintext
# AUTHENTICATION PART
TENABLE_ACCESSKEY="<ACCESSKEY>"
TENABLE_SECRETKEY="<SECRETKEY>"

# HOSTNAME PART
TENABLE_HOSTNAME = "https://tenable.sc"
```
